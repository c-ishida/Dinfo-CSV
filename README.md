# 「お薬の説明」アプリ (CSV版)

漢方薬局ハレノヴァのお客さまがお薬を検索・印刷するためのWebアプリケーションです。

**このリポジトリはCSV版です。** Excel版は [こちら](https://github.com/c-ishida/Dinfo)

## 機能

- お薬の番号で検索（完全一致・部分一致対応）
- 複数キーワードを OR検索（スペースまたはカンマ区切り）
- A5サイズ印刷レイアウト自動生成
- スマホ・タブレット対応
- 全角・半角、ハイフン類の自動正規化

## 使い方

1. https://dinfo-halenova-csv.streamlit.app/ を開く
2. お薬の番号を入力
3. 「検索する」ボタンをクリック
4. 検索結果が表示される
5. 印刷したい場合は、［印刷する］ボタンをクリックする

## 技術スタック

- Python 3.9+
- Streamlit
- pandas
- **データ形式**: CSV (UTF-8)
